
# 📊 Customer Shopping Behavior Analysis

## 🔹 Overview

This project analyzes customer shopping behavior using data analytics techniques. It involves data cleaning, exploratory data analysis (EDA), database integration, and dashboard visualization to extract meaningful insights.

---

## 🔹 Dataset

* Dataset contains customer purchase patterns and behavior
* Includes features like:

  * Customer demographics
  * Purchase history
  * Product categories
* Data was loaded and processed using Python

---

## 🔹 Tools & Technologies

* **Python** (Pandas)
* **MySQL** (Database storage and queries)
* **Power BI** (Dashboard visualization)
* **SQLAlchemy** (Python–MySQL connection)

---

## 🔹 Steps Performed

### 1. Data Loading

* Loaded dataset using Python (Pandas)
* Converted data into structured format

### 2. Data Cleaning

* Removed null values
* Handled duplicates
* Standardized column formats

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer behavior trends
* Identified top products and categories
* Studied spending patterns

### 4. Database Integration

* Created MySQL database: `da_project`
* Stored cleaned dataset into table:

  ```
  customer_shopping_behavior
  ```
* Used SQL queries for analysis

### 5. Power BI Dashboard

* Connected Power BI to MySQL database
* Created interactive dashboards showing:

  * Sales trends
  * Customer segmentation
  * Product performance
